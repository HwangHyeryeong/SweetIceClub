# 🍦 Sweet Ice Club
**프리미엄 아이스크림 `Sweet Ice Club`의 주문 및 재고 관리 시스템** <br>
- 편리한 아이스크림 주문과 결제 경험 제공을 목표로 합니다.
- 빠르고 편리한 주문 프로세스를 통해 고객의 만족도를 높이고, 매출 증가로 이어지는 것을 기대합니다.

## ⏰ 수행 기간
2023.7.31. ~ 2023.8.4. (5일)<br>

**회원 기능 담당**
